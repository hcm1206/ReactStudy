#### 2022-09-18
**Chapter3**\
3.1 ~ 3.6(p.96~p.114)
- JSX 정의와 개념 : 자바스크립트 확장 문법(JavaScript + XML/HTML)

- JSX를 사용할 때는 HTML과 JavaScript를 섞어서 사용하되 HTML 코드 내에서 JavaScript 데이터가 필요할 때는 중괄호 또는 문자열 기호 이용
- JSX를 이용하여 리액트 컴포넌트 생성 후 화면에 렌더링 실습
